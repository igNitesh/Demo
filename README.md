# Demo - Java Learning Project

A comprehensive Java learning project containing examples and exercises covering core Java concepts.

## Project Structure

```
core_java/
├── array/              - Array operations and LeetCode problems
├── basic_question/     - Basic Java concepts (Fibonacci, ternary operations)
├── inheritance/        - Object-oriented programming with inheritance
├── interface_day10/    - Interface implementation examples
└── object_colab/       - Object collaboration and modal structures
```

## Topics Covered

- **Arrays**: Array manipulation, diagonal traversal, LeetCode problems
- **Basic Concepts**: Fibonacci sequences, ternary operators
- **OOP Basics**: Inheritance, method overriding, class hierarchies
- **Interfaces**: Interface definition and implementation
- **Object Collaboration**: Object composition and collaboration patterns

## Getting Started

1. Clone the repository
2. Navigate to the desired topic directory
3. Compile and run the Java files
